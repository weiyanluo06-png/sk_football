# 生康足球队纪念站

武汉纺织大学生物医学工程与健康学院足球队的记录网站。这里保存建队、训练、比赛、队员和球队生活中的真实照片与资料。

## 球队故事

生康足球队于 2025 年开始组建。最初因为不能再以联院形式参赛，学院人数又少，队伍一度连能否成形都不确定。后来有人持续拉人报名、订队服，队员们一起把人数凑齐；第一次排阵容时，大家在食堂讨论出了最初的 4231 站位。

球队的第一段正式比赛是十一人制纺大杯。首战在雨中以 2:4 对阵管理学院，虽然没有赢球，但球队没有被打散，打出了自己的风采。之后在首届八人制新生杯中，球队最终取得第七名。这份成绩来自紧密赛程里一次次拼抢、解围和互相补位。

材料学院一直是生康足球队的兄弟学院。从球队还没有成形，到成立初期的场边指导和加油，材料学院的学长给过球队很多帮助。这份关系也会继续传给之后加入的新队员。

## 赛事归档

学校每学年有三段学院足球赛事，网站会优先展示最新赛事，并通过索引切换到其他赛事：

- 上学期：十一人制纺大杯
- 上学期：八人制新生杯
- 下学期：八人制纺大杯

当前赛事档案中，最新展示的是 `2026 春季 · 八人制纺大杯`；也可切换查看 `2025 秋季 · 八人制新生杯` 和 `2025 秋季 · 十一人制纺大杯`。

## 网站内容

- 首页轮播：球队合照，以及建队和材料学院兄弟情谊的说明
- 赛事档案：按赛事切换对手、比分、进球记录与已知比赛说明
- 队员档案：球员数据、随机展示的三张队员卡与阵容调换预览
- 沉浸式阵容：每次刷新按位置随机生成首发，滚动时依次聚焦前锋、中场、后卫与门将
- 影像墙：雨天训练、比赛围圈、球队合照、毕业和场外日常
- 队史节点：建队、十一人制首战、首届新生杯第七名与材料学院的帮助

## 更新球队内容

### 更新球员资料

编辑 `js/team-data.js`。例如，为队员绑定本地照片：

```js
photo: 'assets/players/wuyuze.jpg'
```

球员照片放在 `assets/players/`，建议使用竖版 JPG 或 WebP。

### 更新首页或影像墙照片

将照片放到 `assets/photos/`，再修改对应路径：

- 首页轮播：编辑 `index.html` 中的 `.hero__slide` 背景图片
- 影像墙：编辑 `js/gallery-data.js` 中的 `image` 字段

建议使用新的文件名，例如 `team-photo-2026.jpg`，避免手机浏览器继续使用旧缓存。

### 更新比赛与荣誉

在 `js/team-data.js` 中维护：

- `competitions`：赛事索引、展示顺序与赛事说明，最新赛事应放在第一项
- `matches`：比赛记录，使用对应的 `competition` 字段归档
- `honors`：真实队史节点
- `startingLineup`：场上位置坐标；页面加载后会按位置随机分配队员
